структура файла users.txt:
Login|HashPassword|Role|Access;
Login - string
HashPassword - string
Role - bool (1 - admin, 0 - user)
Access - bool (1 - admin allows access to the system 0 - admin doesn't allow access to the system)

структура файла employee.txt:
ID|Name|Surname|MiddleName|tableNumber|Year|Month|CountOfHours|Rate;
ID - int
Name - string
Surname - string
MiddleName - string
tableNumber - int
Year - int (year >= 0)
Month - int (1 <= month <= 12)
CountOfHours - double
Rate - double
